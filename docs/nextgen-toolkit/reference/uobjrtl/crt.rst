|uspark| |uobj| C Runtime Library
=================================

The |uberspark| |uobj| C runtime library consist of a set of standard C runtime type definitions and 
memory and string functionality that a |uobj| can use to implement its functionality. The |uobj|
C runtime library provides the following functions:
