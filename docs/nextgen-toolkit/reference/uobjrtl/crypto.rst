|uspark| |uobj| Crypto Runtime Library
======================================

The |uberspark| |uobj| Crypto runtime library consist of a set of cryptographic type definitions and 
functions related to hashes, MAC, and ciphers, that a |uobj| can use to implement its functionality. The |uobj|
Crypto runtime library provides the following functions:
