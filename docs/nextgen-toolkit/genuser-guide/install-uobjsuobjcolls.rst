Install |uobjs| and |uobjcolls|
===============================

..  note::  This section is still a work-in-progress

- install |uobjs|; for a |uobj| that provides a certain functionality that can be used by |uobjcolls|
- install |uobjcolls|; for |uobjcolls| that provide certain functionality for other |uobjcolls|
- install |coss| application bundled with |uobjcolls|
- execution with uberspark enforce desired guarantees/protections, gracefully degrade if uberspark root-of-trust/loaders are not found
