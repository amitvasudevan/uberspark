|uspark| Next-gen Toolkit Preface
=================================

|uspark| next-generation toolkit is currently in alpha stage and experimental. 
The next-generation toolkit is under active development and at some point 
in the near future, will replace the existing toolkit. 

The next-generation toollkit documentation currently comprises of three 
overarching guides: the system integrator's guide, 
the |coss| developer's guide and the contributor's guide.
